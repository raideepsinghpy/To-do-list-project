📝 To-Do List Web App

A simple yet functional To-Do List application built using HTML, CSS, and Vanilla JavaScript.
This project was created as part of my learning journey in web development, where the main goal was learning by building, not just watching tutorials.
Live link: https://raideepsinghpy.github.io/To-do-list-project/

🚀 Features
Add new tasks
Delete completed tasks
Clean and minimal UI
Fully responsive (works on mobile & desktop)
Instant UI updates using DOM manipulation

🛠️ Tech Stack
HTML – Structure
CSS – Styling & responsiveness
JavaScript – Logic, DOM manipulation & event handling

📚 What I Learned
While building this project, I learned:
How to structure a clean UI
Real-world DOM manipulation
Handling user events (clicks, inputs)
Debugging UI & logic issues
Writing cleaner and more readable JavaScript
Making layouts responsive without frameworks

💡 Project Philosophy

This is a basic project, but it represents an important step in my journey.
Instead of copy-pasting code, I focused on understanding each line and fixing issues myself.
